# Snakes and Ladder game
## Assumptions
- Board size is 10x10
- Snaks positions (start and end) are
-- 98 - 1
-- 75 - 42
-- 51 - 27
-- 43 - 21
-- 15 - 8
- Ladder positions (start and end) are
-- 8 - 34
-- 52 - 86
-- 74 - 99

## Winner
-- The user who reaches 100 first will be the winner of the game
